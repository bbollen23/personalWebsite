Research

Topological Analysis of Large-Scale, Multifaceted Data

I currently being advised by Professor Joshua Levine of the Computer Science department on a project intended to study large-scale data sets using topological methods. The goal is to examine existing tools that are used to measure distance between topological structures (bottleneck distance for Persistence Diagrams, interleaving distance of Reeb Graphs, for example) with the intent to apply these metrics or modify them in a way that they become useful to the study of these data sets. For example, suppose we have $n$ data sets from $n$ runs of a climate simulation. Given the stochastic nature of these simulations, many of the features - such as minima and maxima - may be different. How do we measure how similar two of these simulations are to each other? How do we determine what would be the true nature of this data we are attempting to simulate? One initial idea is to measure the pairwise similarity of these structures to each other by computing a "distance" between these simulations.

In this formulation, there are multiple challenges to overcome: What does the distance metric between the topological structures tell us about the underlying scalar field that it arises from? How can we efficiently compute this distance so that the run time of ${n \choose 2}$ computations is reasonable?

Projects

MuView

MuView is an in-development visualization tool to aid in the exploratory analysis of music review data. It pulls review data from multiple music publication websites to create a dynamic, interactive experience for users to navigate the multitude of reviews.

Many current review aggregation sites have no way to make complex queries on the data (such as unions and intersections of sets). Even finding simple lists of the top rated albums by specific publications are difficult. This is an attempt to allow users to navigate the lanscape of review data in a more versatile way. The features include choosing your own publications to pull data from, sort by ratings, peform unions and intersections of multiple sets (groups) of albums, view year-end ranks for each album, and more

MuView is built in React.js and utilizes D3.js for dynamic data interatcions. There will (eventually) be a live-version of this available. The source code for MuView can be seen on GitHub.
